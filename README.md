## 🔧 如何安装

1. 从上方表格中或右侧Releases中选择需要的版本
2. 点击链接下载 APK 文件
3. 在 Android 设备上打开下载的 APK 文件
4. 按照系统提示完成安装，需要开启"允许安装未知来源应用"

⚠️⚠️⚠️注意：多个版本不能共存，下载之前必须删除原版本，
如果原版本已有课表内容，请记得提前导出课表数据！！！
